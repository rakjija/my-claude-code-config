# TODO

## 세션 시작 (필수)

1. README.md TODO 섹션 읽기
2. 미완료 항목 요약해서 사용자에게 보고
3. "어떤 작업부터 할까요?" 질문

## 작업 중

- 새 작업 → TODO 추가 후 진행
- 완료 → 즉시 `[x]` 체크 + 날짜 기록

## 형식

```markdown
### High

- [ ] 작업내용 (@담당자, #이슈, ~기한)
- [x] 완료내용 (@담당자, 완료일)

### Medium

- [ ] 작업내용

### Low

- [ ] 작업내용
```

## 우선순위 기준

- **High**: 긴급, 블로커, 배포 차단
- **Medium**: 일반 기능, 개선
- **Low**: 나중에, nice-to-have
